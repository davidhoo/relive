#### 📌 注意事项
- 精度为城市级别（非街道级别）
- 东京等大城市可能返回区级名称（更精确）
- 最大搜索距离默认 100km（可配置）

#### 💡 建议配置
```yaml
geocode:
  provider: offline       # 主要：离线数据库
  fallback: nominatim    # 备用：在线服务
  cache_enabled: true    # 启用缓存
  cache_ttl: 86400       # 24小时
  offline_max_distance: 100  # 100km
```
