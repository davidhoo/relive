  - [ ] 验证配置正确性

  - [ ] 批量处理（并发控制）
  - [ ] 失败重试
  - [ ] 错误日志
  - [ ] 进度保存

#### 技术要求
- Golang 1.21+
- 复用后端 provider 包
- SQLite 数据库
- 命令行框架（cobra 或 urfave/cli）

#### 使用场景
```bash
# 1. 从 NAS 导出数据
relive export --output /path/to/export.db

# 2. 在任何电脑上分析
relive-analyzer analyze \
  --db export.db \
  --provider ollama \
  --model llava:13b \
  --output analyzed.db

# 3. 导入回 NAS
relive import --file analyzed.db
```
